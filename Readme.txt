Bienvenido a Music Club.

ES

El siguiente proyecto es un formulario para simular el "registro" y "login" en un sitio web de música.
El objetivo es crear un usuario que es almacenado en una base de datos (en este caso he utilizado LocalStorage) para luego dirigirse a la página del "login" y al ingresar mail y contraseña comprobar los datos. Si los mismos coinciden con la base de datos, será dirigido al sitio de bienvenida.

* La opción "Remember me" no se encuentra activa, es solo ilustrativa.

Cuenta con validaciones para todos los campos, por ejemplo:
- Las passwords deben cumplir ciertos requisitos y coincidir.
- Si no son aceptados los términos y condiciones, el formulario no puede ser enviado.
- No se pueden crear dos usuarios con el mismo mail.

Realizado con:
- HTML
- CSS (flexbox)
- JavaScript
- LocalStorage


IT

Benvenuto a Music Club.

Il seguente progetto è un formulario di registro e login per un sito web musicale.
L'obiettivo è creare un utente nel quale i sui dati vengono salvati in un database (in questo caso sarà LocalStorage) per dopo essere indirazzato al sito "login" ed effetuare l'ingreso con l'e-mail e la password create. Se entrambi coincidono con la database, l'utente è indirizzato al sito di benvenuto.

* L'opzione "Remember me" non è attiva, è solo illustrativo.

Il formulario ha delle validazione, per esempio:
- Le password devono soddisfare determinati requisiti e corrispondere.
- In caso di non accettare i termini e condizioni, il formulario non può essere inviato.
- Non si può creare due utenti con gli stessi indirizzi mail.

Realizzato con:
- HTML
- CSS (flexbox)
- JavaScript
- LocalStorage
